# Vyz-Kha Ginger Drink

The Vyz-Kha Ginger Drink is a consumable that increases the Cleric's Constitution by +1 for a week. 2 bottles of Vyz-Kha Ginger Drink can be found across Tolstad: one in a crate behind the Griffon near the Askanii-Reeds Trading Company Building, and one that can be bought from the Potion Seller for 300 Crowns.

The Drink was named after Vyz-Kha.

## In-game Description

A refreshing beverage, blessed by divinity—why else would it burn so good?

## Effect

Consuming the Vyz-Kha Ginger Drink causes the Cleric's Constitution to increase by +1 for a week.
