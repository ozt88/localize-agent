# Vourgeni River Droplets

The Vourgeni River Droplets are a consumable that increases the Cleric's Strength by +1 for a week. 3 bottles of Vourgeni River Droplets can be found across Tolstad: one under a stone in Waterlane that can be detected by Find Secrets, one that can be bought from the Potion Seller for 220 Crowns, and one that can be found in a box in the Guard Tower that can be reached by using Caelum's Mage Hand.

## In-game Description

A vial of blessed river-water from Vourgeni, land of giants. Popular among the Urthguard. It's usually labeled as 'NOT BLOOD'.

## Effect

Consuming the Vourgeni River Droplets causes the Cleric's Strength to increase by +1 for a week.
