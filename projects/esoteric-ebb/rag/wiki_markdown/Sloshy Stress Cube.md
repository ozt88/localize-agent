# Sloshy Stress Cube

The Sloshy Stress Cube is a Trinket.

## In-game Description

A translucent somewhat firm gelatinous cube. These used to be sold as toys/pets. Today they're illegal. Probably because they can technically still dissolve human flesh, albeit very, very slowly.

## Effects

Equipping the Sloshy Stress Cube grants the Cleric +1 to their Intelligence.
