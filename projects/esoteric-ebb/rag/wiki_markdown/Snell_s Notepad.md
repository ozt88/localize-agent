# Snell's Notepad

Snell's Notepad is an equipable Key item held by Snell. It can be pickpocketed at DC 7.

Though it appears as a Key item, it serves no known purpose in the current demo. Previously, there was a line in dialogue with Lady Sageleaf where Snell gets embarrassed for not having it.

## In-game Description

A yellow notepad. It has some writing on it:

Frank, Son of Frank. Main Witness. Need to find.

Nicholas Darrow, motive? Tea shop. Radical activity.

Debt. Bird-owner in trouble?

Freestriders Involved? Obviously.

Explosion: Magic? Powder? Esoteric bullshit? Need cleric.
