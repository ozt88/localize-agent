# Miss Nearly

Miss Nearly is a goblin bard.

Through dialogue with her, it is possible to obtain the option to declare yourself a Bard, providing a new path for the quest Class Conflict to take.

The achievement Tax Deductable is awarded by placing 1,000 crowns in her donation bowl.
