# Tower Key

The Tower Key is an equipable Key Item given by Nicholas Darrow during dialogue. It can also be taken from him without talking to him.

The Tower Key is required to enter the Old Guard Tower.

## In-game Description

An iron key. Clearly dwarven made. While it looks quite old, it's actually extremely sturdy.

It opens the old Guard Tower on Tolstad.

## Effect

Equipping the Tower Key grants the Cleric +1 to their Strength.
