# Sending

Sending is a third level Evocation spell. A weakened version of it can be cast using the Trinket Sending Pebble.

## Effect

Sending can send a message shorter than 30 words between 2 powerful individuals, city states, or arcanists. This ability is not confined by physical distance, but cannot reach a target within a separate plane of existence.

## Trivia

Various Sending Offices employ a number of clerks for the express purpose of casting Sending across the Coast.
