# Esoteric Lockpick

The Esoteric Lockpick is a Trinket located in the South Caverns. It is looted from the body of a headless rogue.

## In-game Description

A throbbing, twitching, electrifying little tool. It yearns for the lock. If you stick it in, it will do the work for you.

## Effects

Equipping the Esoteric Lockpick grants the Cleric +2 to their Dexterity. Additionally, it provides some bonus on checks related to picking locks.
