# Concentrated Rhubarb Ale

Concentrated Rhubarb Ale is a consumable that increases the Cleric's Charisma by +1 for a week. 2 bottle of Concentrated Rhubarb Ale can be found around Tolstad: one can be pickpocketed at DC 15 from Rixbiter, and one can be bought from the Potion Seller for 230 Crowns.

## In-game description

An Askanii classic. Contains a fizzy drink that tastes just like ale - with none of the downsides! So it's actually okay for you to drink this, since it's non-alcoholic.

## Effect

Consuming Concentrated Rhubarb Ale causes the Cleric's Charisma to increase by +1 for a week.
