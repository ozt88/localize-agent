# Warden Kullervo'Lämäläinen

Kullervo is an Elf and a representative of Illym.

## Description

The Warden appears only once, as part of the quest Elven Connection. He claims that there was a plot to destabilize the election, and that Pinja was a part of it.

## Background

The Warden is an Illym Elf, and he serves them loyally.
