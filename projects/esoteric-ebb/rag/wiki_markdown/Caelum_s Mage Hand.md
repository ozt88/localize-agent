# Caelum's Mage Hand

Caelum's Mage Hand is a cantrip that can be obtained from using Comprehend Languages on a spell tome within the upper level of the Visken's Lair. It is a modified version of Mage Hand that has been created by Caelum Diminorii.

## In-game Description

A spectral, floating hand appears at a point you choose within range. You can use your action to control the hand. You can use the hand to manipulate an object, open an unlocked door or container or retrieve an item from an open container. Any trifle check attempted with the hand can be rolled with your intelligence modifier.

The hand can't attack, activate magic items, or carry more than 10 pounds.

## Effects

Caelum's Mage Hand can be used to trifle items that were said to be "slightly out of reach" previously. It can also be used to pickpocket characters, using the Cleric's Intelligence modifier instead of their Dexterity modifier while doing so.

### List of containers/items that can be reached with Mage Hand

* Amulet in bird's nest outside of Visken's Lair
* Vourgeni River Droplets in a box in the Guard Tower
* 300 crowns and a healing potion in a high-up box in Snell's Pad
