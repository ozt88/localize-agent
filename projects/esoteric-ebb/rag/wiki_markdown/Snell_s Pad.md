# Snell's Pad

Snell's Pad is a location in the Holes, a series of stacked underground apartments near Waterlane. It is where Snell lives.

## Description

Upon entering, you may see a box that is on top of dresser - you will need Caelum's Mage Hand to reach it. It contains 300 crowns and a healing potion. Past it is a staircase which features a broken Glowstone Lantern that you can repair. on the right side of the apartment is a loft, which contains Snell's Bed, his information Board, and a box which contains some more of his possessions. It is worth noting that Snell won't prevent you from looting his property.
