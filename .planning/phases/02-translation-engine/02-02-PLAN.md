```go
package clustertranslate

import "localize-agent/workflow/internal/inkparse"

type ClusterTask struct {
    Batch         inkparse.Batch
    PrevGateLines []string       // last 3 lines of previous gate for [CONTEXT] per D-03
    GlossaryJSON  string         // per-batch glossary terms (excluding warmup)
}

type ClusterResult struct {
    BatchID     string
    Lines       []TranslatedLine  // parsed output lines
    RawOutput   string
    Excluded    []string          // block IDs excluded (punctuation-only per D-13)
}

type TranslatedLine struct {
    Number  int    // [NN] marker value
    Speaker string // extracted speaker label (for formatter to strip)
    IsChoice bool  // had [CHOICE] marker
    Text    string // Korean translation text
}

type PromptMeta struct {
    LineCount       int      // number of translatable lines (excluding excluded)
    ExcludedBlockIDs []string // block IDs excluded from prompt
    BlockIDOrder    []string // ordered block IDs matching line numbers
}
```

Create `workflow/internal/clustertranslate/prompt.go`:

- `BuildBaseWarmup(systemPrompt, contextText, rulesText, glossaryWarmupJSON string) string` -- assembles warmup text following v1 `translateSkill.warmup()` pattern. Structure:
  ```
  {systemPrompt}

  {contextText}

  ## Translation Rules
  {v2 static rules - see below}
  {rulesText}

  ## Glossary (preserve these terms in English)
  {glossaryWarmupJSON}

  Reply with exactly: OK
  ```
  v2 static rules (replacing v1 JSON-centric rules):
  1. "Translate the following scene into Korean."
  2. "Maintain the [NN] line numbers exactly as given."
  3. "Do not add, remove, or merge lines."
  4. "Preserve speaker labels (e.g., 'Braxo:') in your output."
  5. "Preserve [CHOICE] markers in your output."
  6. "[CONTEXT] lines are for reference only -- do not translate them."
  7. "All proper nouns (names, places, spells, abilities) stay in English."
  8. "Match the tone and register of the original."
  9. "Output only the translated lines, no commentary."

- `BuildScriptPrompt(task ClusterTask) (string, PromptMeta)` -- builds the batch prompt:
  1. Filter out punctuation-only blocks from task.Batch.Blocks (per D-13: use `isPunctuationOnly` or equivalent). Record excluded IDs in PromptMeta.
  2. If task.PrevGateLines is non-empty, prepend [CONTEXT] block per D-03:
     ```
     [CONTEXT]
     (이전 게이트 -- 번역하지 마세요)
     [C1] "line 1"
     [C2] "line 2"
     [C3] "line 3"

     ---
     ```
  3. For each remaining block, format as:
     - `[NN] Speaker: "block.Text"` if block.Speaker != "" (per D-01)
     - `[NN] [CHOICE] "block.Text"` if block.Choice != "" (per D-02)
     - `[NN] "block.Text"` otherwise
  4. Append per-batch glossary if non-empty: `\n## Batch Glossary\n{task.GlossaryJSON}`
  5. Append content-type suffix rules via `BuildContentSuffix(task.Batch.ContentType)`
  6. Return prompt string and PromptMeta with LineCount, ExcludedBlockIDs, BlockIDOrder

- `BuildContentSuffix(contentType string) string` -- per D-04, type-specific suffix rules:

Create `workflow/internal/clustertranslate/parser.go`:

- `ParseNumberedOutput(raw string) ([]TranslatedLine, error)` -- parse LLM output into TranslatedLine slice:
  1. Split raw output by newlines
  2. For each non-empty line, match regex `^\[(\d+)\]\s*(.*)$`
  3. From the captured text after [NN]:
     - If starts with `[CHOICE]`, set IsChoice=true and strip `[CHOICE]` prefix
     - If contains `:` pattern like `SpeakerName: "..."`, extract Speaker and Text
     - Otherwise Text = remaining content
  4. Strip surrounding quotes from Text if present
  5. Return ordered slice of TranslatedLine

- `MapLinesToIDs(lines []TranslatedLine, meta PromptMeta) (map[string]string, error)` -- per TRANS-03:
  1. Verify len(lines) == len(meta.BlockIDOrder), return error if mismatch
  2. Build map[blockID]koreanText by matching line index to BlockIDOrder index
  3. Return the mapping

- `ValidateTranslation(rawOutput string, meta PromptMeta, sourceTexts []string) error` -- per TRANS-04 and D-13:

  4. Return nil if all checks pass, or first error encountered
