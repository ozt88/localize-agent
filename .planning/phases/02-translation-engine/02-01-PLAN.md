```go
package contracts

type V2PipelineItem struct {
    ID                string   // DialogueBlock.ID (path-based)
    SortIndex         int
    SourceFile        string
    Knot              string
    ContentType       string   // dialogue, spell, ui, item, system
    SourceRaw         string   // original EN text
    SourceHash        string   // SHA-256 of SourceRaw
    HasTags           bool     // whether source contains rich-text tags
    State             string
    KORaw             string   // Stage 1 output (tag-free Korean)
    KOFormatted       string   // Stage 2 output (Korean with tags restored)
    TranslateAttempts int
    FormatAttempts    int
    ScoreAttempts     int
    ScoreFinal        float64
    FailureType       string   // translation/format/both/pass per D-14
    LastError         string
    AttemptLog        string   // JSON array per D-16
    ClaimedBy         string
    BatchID           string   // which Batch this block belongs to
}

type V2PipelineStore interface {
    Seed(items []V2PipelineItem) (inserted int, skipped int, err error)
    ClaimPending(pendingState, workingState, workerID string, batchSize int, leaseSec int) ([]V2PipelineItem, error)
    MarkState(id, newState string) error
    MarkTranslated(id, koRaw string) error
    MarkFormatted(id, koFormatted string) error
    MarkScored(id string, scoreFinal float64, failureType, reason string) error
    MarkFailed(id, lastError string) error
    AppendAttemptLog(id string, entry map[string]interface{}) error
    UpdateRetryState(id, targetState string, incrementField string) error
    CleanupStaleClaims(olderThanSec int) (int64, error)
    CountByState() (map[string]int, error)
    GetItem(id string) (*V2PipelineItem, error)
    Close() error
}
```

```sql
CREATE TABLE IF NOT EXISTS pipeline_items_v2 (
    id TEXT PRIMARY KEY,
    sort_index INTEGER NOT NULL DEFAULT 0,
    source_file TEXT NOT NULL DEFAULT '',

    content_type TEXT NOT NULL DEFAULT '',

    ko_formatted TEXT,
    translate_attempts INTEGER NOT NULL DEFAULT 0,
    format_attempts INTEGER NOT NULL DEFAULT 0,
    score_attempts INTEGER NOT NULL DEFAULT 0,
    score_final REAL NOT NULL DEFAULT -1,
    failure_type TEXT NOT NULL DEFAULT '',
    last_error TEXT NOT NULL DEFAULT '',
    attempt_log JSONB,
    claimed_by TEXT NOT NULL DEFAULT '',
    claimed_at TIMESTAMPTZ,
    lease_until TIMESTAMPTZ,
    batch_id TEXT NOT NULL DEFAULT '',
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
CREATE INDEX IF NOT EXISTS idx_pv2_state ON pipeline_items_v2(state);
CREATE INDEX IF NOT EXISTS idx_pv2_state_lease ON pipeline_items_v2(state, lease_until);
```

Create `workflow/internal/v2pipeline/store.go` implementing `contracts.V2PipelineStore`:

- `Open(dsn string) (*Store, error)` -- opens PostgreSQL connection, runs embedded schema DDL
- `Seed(items []V2PipelineItem) (inserted, skipped int, err error)` -- per INFRA-02: `INSERT INTO ... ON CONFLICT(source_hash) DO NOTHING`. For each item, if `item.State == "done"` (passthrough), insert directly as done. Returns count of inserted vs skipped (already existing).
- `ClaimPending(pendingState, workingState, workerID string, batchSize, leaseSec int) ([]V2PipelineItem, error)` -- follows v1 pattern: `UPDATE ... SET state=$2, claimed_by=$3, claimed_at=NOW(), lease_until=NOW()+interval WHERE state=$1 AND (lease_until IS NULL OR lease_until < NOW()) LIMIT $4 RETURNING *`
- `MarkState(id, newState string) error` -- simple state update + clear claimed_by/lease_until + set updated_at
- `MarkTranslated(id, koRaw string) error` -- set ko_raw, state based on has_tags: if has_tags=true -> "pending_format", else -> "pending_score". Increment translate_attempts.
- `MarkFormatted(id, koFormatted string) error` -- set ko_formatted, state="pending_score", increment format_attempts
- `MarkScored(id string, scoreFinal float64, failureType, reason string) error` -- set score_final, failure_type. Per D-14: "pass"->done, "translation"->pending_translate, "format"->pending_format, "both"->pending_translate. Increment score_attempts.
- `MarkFailed(id, lastError string) error` -- state="failed", last_error=lastError
- `AppendAttemptLog(id string, entry map[string]interface{}) error` -- append JSON object to attempt_log JSONB array: `UPDATE ... SET attempt_log = COALESCE(attempt_log, '[]'::jsonb) || $2::jsonb`
- `UpdateRetryState(id, targetState, incrementField string) error` -- set state=targetState, increment the specified attempts field, clear claimed_by
- `CleanupStaleClaims(olderThanSec int) (int64, error)` -- find rows where state starts with "working_" and lease_until < NOW()-interval, reset to corresponding pending state
- `CountByState() (map[string]int, error)` -- `SELECT state, COUNT(*) FROM pipeline_items_v2 GROUP BY state`
- `GetItem(id string) (*V2PipelineItem, error)` -- single row lookup by id
- `Close() error`

Create `workflow/cmd/go-v2-ingest/main.go`:

CLI flags:
- `-input` (required): path to Phase 1 parser JSON output (array of ParseResult)
- `-dsn` (required): PostgreSQL DSN for pipeline_items_v2
- `-backend` (default "postgres"): DB backend
- `-db` (default ""): SQLite DB path (for local dev)

Flow:
1. Read input JSON file -> decode as `[]inkparse.ParseResult`
2. Call `inkparse.BuildBatches(results)` to get batch assignments
3. Build a map from block ID -> batch ID for batch_id field
4. For each block across all ParseResults:
   - Create `contracts.V2PipelineItem` with:
     - ID = block.ID
     - SortIndex = global sequential counter
     - SourceFile = block.SourceFile
     - Knot = block.Knot
     - ContentType = block.ContentType
     - SourceRaw = block.Text
     - SourceHash = block.SourceHash
     - HasTags = detect via `regexp.MustCompile(`<[a-zA-Z][^>]*>`)` on block.Text
     - State = if block.IsPassthrough then "done" else "pending_translate"
     - KOFormatted = if block.IsPassthrough then block.Text else ""
     - BatchID = lookup from batch map
5. Call `store.Seed(items)` in chunks of 500
6. Print summary: total blocks, inserted, skipped (dedup), passthrough (done immediately)

Per INFRA-02: the Seed method handles dedup via ON CONFLICT(source_hash) DO NOTHING. The CLI does NOT do blind INSERT.

Per Pitfall 6 from RESEARCH.md: passthrough blocks get state="done", ko_formatted=source_raw immediately.
