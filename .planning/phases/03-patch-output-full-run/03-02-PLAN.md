Critical pattern for injection:
- Parser walkFlatContent collects "^text" strings into textBuf
- On flush, textBuf content becomes block.Text and SourceHash(text) is computed
- Injection must: walk same path, collect same "^text" nodes, compute same hash,
  then REPLACE the "^text" node values with Korean translation
- Strategy (per RESEARCH.md Pitfall 1): first "^text" node gets "^" + full Korean text,
  remaining nodes in same block get "^" (empty text prefix, preserving node count)

    - json.MarshalIndent not needed -- json.Marshal is fine since ink JSON is not human-edited
