### 7. TryTranslate Chain (~120 lines)

Per D-05 (modified by Open Question 2 resolution — GeneratedPattern moved to RuntimeLexicon):

**4-stage chain (NOT 5 — GeneratedPattern regex merged into RuntimeLexicon):**

```csharp
internal static bool TryTranslate(ref string value, string origin = "unknown")
{
    if (string.IsNullOrEmpty(value)) return false;
    var original = value;

    // Stage 1: Exact dictionary lookup (TranslationMap)
    if (TranslationMap.TryGetValue(value, out var t) && !string.IsNullOrEmpty(t))
    { value = t; RecordHit("exact", original, t); return true; }

    // Stage 2: DC/FC prefix strip -> lookup body in TranslationMap
    if (TryTranslateDcFcStrip(ref value, original))
    { RecordHit("dcfc_strip", original, value); return true; }

    // Stage 3: Contextual (source_file + history scoring)
    if (TryTranslateContextual(ref value, original))
    { RecordHit("contextual", original, value); return true; }

    // Stage 4: Runtime lexicon (exact + substring + regex)
    if (TryTranslateRuntimeLexicon(ref value))
    { RecordHit("lexicon", original, value); return true; }

    _misses++;
    CaptureUntranslated(original, origin);
    return false;
}
```

**TryTranslateDcFcStrip(ref string value, string original):**
- Regex `^[A-Z]{2}\d+\s+\w+-` (must match export.go dcfcPrefixRe exactly)
- Strip prefix, look up body in TranslationMap
- If found, set value to translation

**TryTranslateContextual(ref string value, string original):**
- Port from v1 lines 2203-2275 with simplification.
- Lookup ContextualMap by value (source text). Get List<ContextualEntry>.
- If 1 candidate: use it directly.
- If multiple: score by _currentDialogSourceFile match (highest priority), then _recentContextHistory overlap.
- NO StripGameplayPrefix (v2 sources are already clean). NO NormalizeKey.

**TryTranslateRuntimeLexicon(ref string value):**
- Order: exact -> substring (longest-match-first) -> regex
- Exact: RuntimeExactReplacements.TryGetValue
- Substring: iterate RuntimeSubstringReplacements, value.Contains(find) -> value.Replace(find, replace)
- Regex: iterate RuntimeRegexRules, rule.Pattern.IsMatch(value) -> rule.Pattern.Replace(value, rule.ReplaceString)
