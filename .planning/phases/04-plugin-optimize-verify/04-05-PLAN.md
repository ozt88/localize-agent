    **Part B: PLUGIN-03 verification (95%+ coverage)**

    Read translation_loader_state.json. Calculate coverage using PROPER methodology:

    The game engine re-submits already-translated text through TryTranslate after wrapping it in TMP rendering tags (`<color>`, `<link>`, `<line-indent>`, `<#FFFFFFFF>`). These are NOT real misses. To calculate real coverage:

    1. Read untranslated_capture.json -- these are the UNIQUE strings that went through TryTranslate and found no match
    2. Filter out entries that contain Korean characters (already translated, re-wrapped)
    3. Filter out entries that are pure TMP markup (no meaningful text)
    4. Count remaining entries as genuine misses
    5. Compare against total unique text captured in full_text_capture.json

    ```python
    import json, re

    # Load capture data
    state = json.load(open('E:/SteamLibrary/steamapps/common/Esoteric Ebb/BepInEx/translation_loader_state.json'))
    untrans = json.load(open('E:/SteamLibrary/steamapps/common/Esoteric Ebb/BepInEx/untranslated_capture.json'))
    capture = json.load(open('E:/SteamLibrary/steamapps/common/Esoteric Ebb/BepInEx/full_text_capture.json'))

    # Total unique text seen
    total_unique = capture.get('count', len(capture.get('entries', [])))

    # Untranslated analysis
    entries = untrans.get('entries', untrans if isinstance(untrans, list) else [])
    korean_re = re.compile(r'[\uAC00-\uD7AF\u3130-\u318F]')
    real_misses = []
    korean_rewrap = []
    for e in entries:
        src = e.get('source', e.get('text', ''))
        if korean_re.search(src):
            korean_rewrap.append(src)
        else:
            real_misses.append(src)

    print(f'=== PLUGIN-03 Coverage Analysis ===')
    print(f'Translations loaded: {state["translations_loaded"]}')
    print(f'Total unique text captured: {total_unique}')
    print(f'Untranslated entries: {len(entries)}')
    print(f'  - Korean re-wrap (not real miss): {len(korean_rewrap)}')
    print(f'  - Genuine misses: {len(real_misses)}')
    print(f'')
    print(f'Coverage = (total_unique - genuine_misses) / total_unique')
    if total_unique > 0:
        coverage = (total_unique - len(real_misses)) / total_unique * 100
        print(f'Coverage: {coverage:.1f}%')
        print(f'Target: 95%+  {"PASS" if coverage >= 95 else "FAIL"}')
    print(f'')
    print(f'=== Top 20 Genuine Misses ===')
    for m in real_misses[:20]:
        print(f'  {m[:120]}')
    ```
