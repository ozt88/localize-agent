### Task 2: Before/After Diff 생성

**Files:**
- `projects/esoteric-ebb/tools/diff_translations.py` (CREATE 또는 MODIFY)

**Action:** CREATE/MODIFY

**2-1. diff 도구 작성 (또는 기존 도구 활용):**

`diff_translations.py` — 두 translations.json 파일을 비교하여:
- 변경된 항목 수 (before != after)
- 추가된 항목 수 (after에만 존재)
- 제거된 항목 수 (before에만 존재)
- 샘플 변경 10건 출력 (before → after)

```bash
python3 projects/esoteric-ebb/tools/diff_translations.py \
  "E:/SteamLibrary/steamapps/common/Esoteric Ebb/BepInEx/plugins/TranslationLoader/translations.json.before-phase09" \
  "E:/SteamLibrary/steamapps/common/Esoteric Ebb/BepInEx/plugins/TranslationLoader/translations.json"
```

**2-2. Kattegatt before/after 비교:**
diff 출력에서 Kattegatt 관련 항목 필터링:
- before: `너/~다` 현대 반말 (Phase 08 결과)
- after: `그대/~도다/~노라` 고어체 (복원 후 결과)

**Verification:**
- diff 스크립트 실행 성공

- Kattegatt 항목에서 고어체 복원 확인
