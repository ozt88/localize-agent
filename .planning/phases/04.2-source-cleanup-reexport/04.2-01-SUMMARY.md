## Decisions Made
- **Collision handling strategy:** 5,586 rows had DC/FC prefixes but only 537 had unique clean bodies. The other 5,049 stripped to bodies already existing in the DB (duplicate source_raw). Rather than aborting, the migration skips collision rows. Those 101 remaining DC/FC-prefixed rows in the DB are duplicates whose clean bodies are already served by other entries.
- **Parser stores ability type in Tags:** When stripping "DC12 str-", the ability type "str" is preserved in the block's Tags array (not lost).
